# Phase2-02: スプレッドシート・GAS仕様

**バージョン**: 1.0
**作成日**: 2026-04-21
**ベース**: `仕様元/スクリーニングスプレッドシート_GAS設計書_v1.0.md` を新DB設計に合わせて更新

---

## このドキュメントの役割

スカウトが作業するGoogleスプレッドシートの構成と、「📤 Notionへ転記」ボタンの実装（GAS）仕様。スプレッドシート作成担当・GAS実装担当（Claude Code）が読む。

---

## 1. 概要

候補者スクリーニングをNotionを開かずGoogleスプレッドシート内で完結させる。「○」判定されたレコードはGASで自動的にNotionマスターDBへ転記される。

```
┌────────────────────────────────────────────────┐
│ 「候補プール」シート                            │
│                                                │
│  [Claude Codeが夜間バッチで候補を追加]          │
│                                                │
│  行1: 候補者A  [○/保留/×] [S/A/B/C] [コメント] │
│  ...                                           │
│                                                │
│  [📤 Notionへ転記ボタン]                        │
└───────────────┬────────────────────────────────┘
                ▼
┌────────────────────────────────────────────────┐
│ GAS                                            │
│  1. ○判定かつ未転記のレコード抽出              │
│  2. Notionマスターに新規ページ作成             │
│     マスターステータス=候補、ランク=仮ランク    │
│  3. スプレッドシート側を「転記済み」に更新     │
└────────────────────────────────────────────────┘
```

## 2. スプレッドシート構成

### 2-1. ファイル情報

| 項目 | 値 |
|---|---|
| ファイル名 | `イラストレーター候補スクリーニング` |
| オーナー | 加藤（Roadie） |
| 共有設定 | スカウト: 編集権限 / オーナー: 表示権限 |
| サービスアカウント | 編集者として共有必須（illustrator-scraper用） |

### 2-2. シート構成

| シート名 | 用途 | 主担当 |
|---|---|---|
| **候補プール** | メインの作業シート | スカウト |
| 転記ログ | Notion転記の実行履歴 | GAS自動記録 |
| 設定 | GAS動作設定値 | オーナー |
| README | スカウト向け簡易マニュアル | ー |

### 2-3. 「候補プール」シートの列定義

| 列 | ヘッダー | 型 | 入力者 | 説明 |
|---|---|---|---|---|
| A | 検出日 | DATE | 自動 | バッチ実行日 |
| B | 検出元 | TEXT | 自動 | シードXアカウント（例: `@example_artist`） |
| C | Xアカウント | URL | 自動 | `https://x.com/username` |
| D | 表示名 | TEXT | 自動 | X上の表示名 |
| E | プロフィール | TEXT | 自動 | bio（最大500文字） |
| F | フォロワー数 | NUMBER | 自動 | |
| G | Pixivリンク | URL | 自動 | bioから抽出 |
| H | ポートフォリオ | URL | 自動 | lit.link・Potofu等 |
| I | 既存DB重複 | TEXT | 自動 | `YES` / `NO` |
| **J** | **判定** | **SELECT** | **スカウト** | **`○` / `保留` / `×` / （空）** |
| **K** | **画力ランク候補** | **SELECT** | **スカウト** | **`S` / `A` / `B` / `C` / （空）** |
| **L** | **コメント** | **TEXT** | **スカウト** | **自由記述** |
| M | 転記状態 | TEXT | GAS | `未転記` / `転記済み` / `転記失敗` |
| N | 転記日時 | DATETIME | GAS | |
| O | NotionページID | TEXT | GAS | |

### 2-4. データ検証（プルダウン）

- **J列**: `○`、`保留`、`×`
- **K列**: `S`、`A`、`B`、`C`
- **M列**: `未転記`（初期値）、`転記済み`、`転記失敗`

### 2-5. 条件付き書式

| 条件 | 書式 |
|---|---|
| J列=`○` | 背景：薄い緑 |
| J列=`×` | 背景：薄い灰色、文字：取り消し線 |
| J列=`保留` | 背景：薄い黄色 |
| I列=`YES` | 背景：薄いオレンジ |
| M列=`転記済み` | 背景：薄い青 |
| M列=`転記失敗` | 背景：薄い赤 |

### 2-6. 固定・保護

- 1行目（ヘッダー）: 固定
- A〜I列・M〜O列: スカウト編集不可（GAS・Claude Codeのみ書き込み）
- J〜L列: スカウトが自由に編集可

### 5-2. main.gs

```javascript
/**
 * スプレッドシート起動時にメニュー追加
 */
function onOpen() {
  const ui = SpreadsheetApp.getUi();
  ui.createMenu('🔧 運用ツール')
    .addItem('📤 Notionへ転記', 'transferToNotion')
    .addSeparator()
    .addItem('🔍 Notion疎通テスト', 'testNotionConnection')
    .addItem('📊 転記ログを表示', 'showTransferLog')
    .addToUi();
}

/**
 * メインの転記処理
 */
function transferToNotion() {
  const ui = SpreadsheetApp.getUi();
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('候補プール');
  const config = loadConfig();

  const targets = findTransferTargets(sheet);
  if (targets.length === 0) {
    ui.alert('転記対象なし', '○判定かつ未転記のレコードがありません。', ui.ButtonSet.OK);
    return;
  }

  const response = ui.alert(
    '転記確認',
    `${targets.length}件のレコードをNotionへ転記します。よろしいですか？`,
    ui.ButtonSet.OK_CANCEL
  );
  if (response !== ui.Button.OK) return;

  const results = { success: 0, failure: 0, errors: [] };

  targets.forEach(target => {
    try {
      const notionPageId = createNotionPage(target, config);
      markAsTransferred(sheet, target.rowIndex, notionPageId);
      results.success++;
    } catch (err) {
      markAsFailed(sheet, target.rowIndex, err.message);
      results.failure++;
      results.errors.push({
        row: target.rowIndex,
        xUsername: target.xUsername,
        error: err.message
      });
    }
    Utilities.sleep(400); // Notion API: 3 req/sec 制限
  });

  writeTransferLog(results, targets.length);

  ui.alert(
    '転記完了',
    `成功: ${results.success}件 / 失敗: ${results.failure}件\n\n詳細は「転記ログ」シートを確認してください。`,
    ui.ButtonSet.OK
  );
}

/**
 * Notion 疎通テスト
 */
function testNotionConnection() {
  const ui = SpreadsheetApp.getUi();
  const config = loadConfig();
  try {
    const result = queryNotionDataSource(config.notionApiKey, config.dataSourceId, { page_size: 1 });
    ui.alert('疎通成功', `Notion APIへの接続に成功しました。\nDB内レコード数（最新1件のみ取得）: ${result.results.length}`, ui.ButtonSet.OK);
  } catch (err) {
    ui.alert('疎通失敗', `エラー: ${err.message}`, ui.ButtonSet.OK);
  }
}
```

```javascript
/**
 * Notionマスターに新規ページ作成
 */
function createNotionPage(target, config) {
  const url = 'https://api.notion.com/v1/pages';

  const properties = {
    '作家名': {
      title: [{ text: { content: target.displayName || target.xUsername } }]
    },
    'Xリンク': { url: target.xUrl },
    ...(target.pixivUrl ? { 'Pixivリンク': { url: target.pixivUrl } } : {}),
    ...(target.portfolioUrl ? { 'その他連絡先': { url: target.portfolioUrl } } : {}),
    '検出元アカウント': {
      rich_text: [{ text: { content: target.detectedFrom || '' } }]
    },
    ...(target.powerRankCandidate ? {
      'ランク': { select: { name: target.powerRankCandidate } }
    } : {}),
    'マスターステータス': {
      status: { name: config.initialStatus }  // デフォルト: 候補
    },
    '見つけた日': {
      date: { start: formatDate(new Date()) }
    },
    '見つけた人': {
      select: { name: config.foundBy }  // デフォルト: Claude Code
    },
    ...(target.screenerComment ? {
      '備考': {
        rich_text: [{ text: { content: `[スカウト] ${target.screenerComment}` } }]
      }
    } : {})
  };

  const payload = {
    parent: { data_source_id: config.dataSourceId },
    properties: properties
  };

  const options = {
    method: 'post',
    contentType: 'application/json',
    headers: {
      'Authorization': `Bearer ${config.notionApiKey}`,
      'Notion-Version': '2025-09-03'
    },
    payload: JSON.stringify(payload),
    muteHttpExceptions: true
  };

  const response = UrlFetchApp.fetch(url, options);
  const code = response.getResponseCode();
  const body = JSON.parse(response.getContentText());

  if (code !== 200 && code !== 201) {
    throw new Error(`Notion APIエラー (${code}): ${body.message || '不明なエラー'}`);
  }

  return body.id;
}

function queryNotionDataSource(apiKey, dataSourceId, options = {}) {
  const url = `https://api.notion.com/v1/data_sources/${dataSourceId}/query`;
  const fetchOptions = {
    method: 'post',
    contentType: 'application/json',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Notion-Version': '2025-09-03'
    },
    payload: JSON.stringify(options),
    muteHttpExceptions: true
  };
  const response = UrlFetchApp.fetch(url, fetchOptions);
  const code = response.getResponseCode();
  const body = JSON.parse(response.getContentText());
  if (code !== 200) {
    throw new Error(`Notion APIエラー (${code}): ${body.message || '不明なエラー'}`);
  }
  return body;
}
```

### 5-4. sheet.gs

```javascript
/**
 * 転記対象を抽出（J列=○ かつ M列=未転記）
 */
function findTransferTargets(sheet) {
  const lastRow = sheet.getLastRow();
  if (lastRow < 2) return [];

  const data = sheet.getRange(2, 1, lastRow - 1, 15).getValues();
  const targets = [];

  data.forEach((row, idx) => {
    const rowIndex = idx + 2;
    const judgment = row[9];   // J列
    const status = row[12];    // M列

    if (judgment === '○' && status === '未転記') {
      targets.push({
        rowIndex: rowIndex,
        detectedDate: row[0],
        detectedFrom: row[1],
        xUrl: row[2],
        displayName: row[3],
        profile: row[4],
        followerCount: row[5],
        pixivUrl: row[6],
        portfolioUrl: row[7],
        isDuplicate: row[8] === 'YES',
        xUsername: extractUsername(row[2]),
        powerRankCandidate: row[10],
        screenerComment: row[11]
      });
    }
  });

  return targets;
}

function markAsTransferred(sheet, rowIndex, notionPageId) {
  sheet.getRange(rowIndex, 13).setValue('転記済み');
  sheet.getRange(rowIndex, 14).setValue(new Date());
  sheet.getRange(rowIndex, 15).setValue(notionPageId);
}

function markAsFailed(sheet, rowIndex, errorMessage) {
  sheet.getRange(rowIndex, 13).setValue('転記失敗');
  sheet.getRange(rowIndex, 12).setValue(
    (sheet.getRange(rowIndex, 12).getValue() || '') + ` [ERR: ${errorMessage}]`
  );
}

function extractUsername(xUrl) {
  const match = String(xUrl).match(/(?:twitter|x)\.com\/([^\/\?]+)/);
  return match ? match[1] : '';
}

function writeTransferLog(results, targetCount) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const logSheet = ss.getSheetByName('転記ログ') || ss.insertSheet('転記ログ');
  if (logSheet.getLastRow() === 0) {
    logSheet.getRange(1, 1, 1, 6).setValues([[
      '実行日時', '実行者', '対象件数', '成功件数', '失敗件数', '詳細ログ'
    ]]);
  }
  const errorDetail = results.errors
    .map(e => `行${e.row}(@${e.xUsername}): ${e.error}`)
    .join('\n');
  logSheet.appendRow([
    new Date(),
    Session.getActiveUser().getEmail(),
    targetCount,
    results.success,
    results.failure,
    errorDetail
  ]);
}
```

```javascript
/**
 * 設定シート＋スクリプトプロパティから設定読み込み
 */
function loadConfig() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('設定');
  if (!sheet) throw new Error('「設定」シートが存在しません');

  const scriptProperties = PropertiesService.getScriptProperties();

  return {
    notionApiKey: scriptProperties.getProperty('NOTION_API_KEY')
                  || sheet.getRange('A2').getValue(),
    dataSourceId: sheet.getRange('A3').getValue(),
    initialStatus: sheet.getRange('A4').getValue() || '候補',
    foundBy: sheet.getRange('A5').getValue() || 'Claude Code',
    batchSize: sheet.getRange('A6').getValue() || 50
  };
}
```

```javascript
function formatDate(date) {
  const y = date.getFullYear();
  const m = String(date.getMonth() + 1).padStart(2, '0');
  const d = String(date.getDate()).padStart(2, '0');
  return `${y}-${m}-${d}`;
}

function showTransferLog() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('転記ログ');
  if (sheet) {
    SpreadsheetApp.setActiveSheet(sheet);
  } else {
    SpreadsheetApp.getUi().alert('「転記ログ」シートがまだありません。');
  }
}
```

## 6. 初回セットアップ手順

### 6-1. スプレッドシート作成

1. 新規Googleスプレッドシート作成
2. ファイル名を `イラストレーター候補スクリーニング` に変更
3. 4つのシート作成: `候補プール`、`転記ログ`、`設定`、`README`
4. 「候補プール」にA〜O列のヘッダー入力
5. J・K列にデータ検証（プルダウン）設定
6. 条件付き書式設定（§2-5）

### 6-2. Notion Integration

1. https://www.notion.so/profile/integrations
2. 「New integration」→ 内部インテグレーション作成
3. Secret をコピー
4. Notionのイラストレーター情報DBページで「•••」→「接続」→ 作成したインテグレーション追加

### 6-3. Google Apps Script

1. 「拡張機能」→「Apps Script」
2. 5つのファイル（main.gs, notion.gs, sheet.gs, config.gs, utils.gs）作成・コピペ
3. 「プロジェクト設定」→「スクリプトプロパティ」
   - Key: `NOTION_API_KEY`
   - Value: Notion Integration Secret
4. 設定シートA3にデータソースIDを入力

### 6-4. サービスアカウント共有

illustrator-scraperのGCPサービスアカウントメールを「編集者」として共有。

### 6-5. 動作確認

1. 手動で1行ダミーデータを入力、J列に`○`
2. 「🔧 運用ツール」→「🔍 Notion疎通テスト」
3. 「📤 Notionへ転記」実行
4. Notionマスターに作成されたか確認

## 7. 運用ルール

### スカウト

- 週1回、50件目安で判定
- 判定後は必ず「📤 Notionへ転記」押下
- 迷ったら `保留`
- A〜I列・M〜O列は編集禁止

### オーナー

- 毎週「保留」一覧を確認し追加判定
- GASエラーレコードは手動対応

### メンテナンス

- 四半期に1回、転記ログをチェック
- 古い `転記済み`（3ヶ月以上）は別シートへアーカイブ
- Notion Integration Secretは半年に1回ローテーション

## 8. トラブルシューティング

| 症状 | 対処 |
|---|---|
| メニューが出ない | ページ再読み込み |
